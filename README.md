# tg-bot-pi
Bot for Telegram (tg)

## requirements
* Need to install [telegram-cli](https://github.com/vysheng/tg) first.
* Change directory absolute **PATH**

## usage
```
telegram-cli -W -s <path/to/action/script>
```
