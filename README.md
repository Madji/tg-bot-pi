# tg-bot-pi
Bot for Telegram (tg)

## requirements
* Need to install [telegram-cli](https://github.com/vysheng/tg) first.
* Change absolute path for **HOME** and **TMP_PATH**

## usage
```
telegram-cli -W -s <path/to/script/action.lua>
```
